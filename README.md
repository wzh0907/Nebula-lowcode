## 简介
#### 星云座低代码是一款应用系统功能扩展平台，它连接应用系统数据库和接口、提供低代码开发能力、打通系统数据、业务和用户，快速扩展新功能，并融入原有系统。星云座采用了独特的前后端分离开发模式，遵循了可视化IDE的设计理念，可与原生代码开发混合使用。
